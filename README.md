# 🏟️ Stade Souk

> Stadium marketplace PWA delivering food and drinks directly to your seat during live games

Built for **Info Days Hackathon v2** (2030 World Cup Morocco theme)

## 🎯 The Problem

Stadium fans face long concession lines, missing crucial game moments just to grab food or drinks. Current solutions force fans to choose between enjoying the game and staying fed.

## 💡 Our Solution

Stade Souk is a Progressive Web App that brings the marketplace to your seat:
- Order from your phone without leaving your seat
- Delivery riders bring orders directly to your section
- Real-time tracking so you know when to expect your order

## ✨ Key Features

### For Fans (Users)
- 🎫 **Smart Seat Detection** - Automatically extracts your seat and section from your ticket
- 💬 **Natural Language Ordering** - Say "I'm thirsty" and we'll show you drinks
- 📱 **PWA Technology** - Works like a native app, no download required
- 🔔 **Order Tracking** - Real-time updates on your order status

### For Vendors (Admins)
- 📊 **Live Dashboard** - See incoming orders in real-time
- 🤖 **AI Order Prediction** - ML model predicts order volume to optimize inventory
- 📈 **Analytics** - Track popular items and peak ordering times

### For Delivery (Riders)
- 🗺️ **Optimized Routes** - AI-calculated paths to minimize delivery time
- 📦 **Order Queue** - Pick up ready orders and mark as delivered
- 🎯 **Section Navigation** - Clear directions to each seat location

## 🛠️ Tech Stack

**Frontend**
- React.js
- Progressive Web App (PWA)
- Real-time WebSocket connections

**Backend**
- Node.js
- Express.js
- Real-time order synchronization

**AI/ML**
- Order volume prediction model
- Natural language processing for user queries
- Route optimization algorithms

## 🚀 Getting Started

### Prerequisites
- Node.js 14+ 
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/JSN-Y/Stadium-EATS.git
cd Stadium-EATS
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm run dev
```

4. Open http://localhost:3000 in your browser
